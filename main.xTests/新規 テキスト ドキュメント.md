C#でコーディングしています。
xUnitで単体テストをしようとしていますが、デザインパターンでいうFactoryパターンのような方法でインスタンス化するメソッドをテストしたいです。

たとえば以下のようなメソッドです。

```csharp
public static class SampleFactory{
    public static IProduct Create( EnumTypes et )
    {
        if( et == EnumTypes.One ) return new Product1();
        if( et == EnumTypes.Two ) return new Product2();
        if( et == EnumTypes.Three ) return new Product3();
        throw new Exception( "存在しない定義です" ); // いずれにも該当しなければ例外を投げる
    }
}
```

テスト自体は「`EnumTypes.One`を渡したら`Product1`のインスタンスが返ってくる」といったテストと、「`EnumTypes.Unknown`を渡したら例外が投げられる」というテストの4通りのテストを行うのだと思いますが、書き方がわかりません…。
